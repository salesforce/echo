# echo
Invalidate-able transparent reverse proxy cache 

### How to use?

#### Requirements
OpenSSL and PCRE libraries are required.

Mac OS:

```bash
$ brew install openssl pcre
```

Debian/Ubuntu:

```bash
$ apt-get install openssl libssl-dev libpcre3
```

To build:

```bash
$ ./bang.sh compile
```

To run:

```bash
$ ./bang.sh run
```
